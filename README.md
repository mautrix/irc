# mautrix-irc
A Matrix-IRC puppeting bridge.

This is currently at a very early stage, so switching from Heisenbridge is not
yet recommended.

## Documentation
All setup and usage instructions are located on [docs.mau.fi]. Some quick links:

[docs.mau.fi]: https://docs.mau.fi/bridges/go/irc/index.html

  (or [with Docker](https://docs.mau.fi/bridges/general/docker-setup.html?bridge=irc))
* Basic usage: [Authentication](https://docs.mau.fi/bridges/go/irc/authentication.html)
